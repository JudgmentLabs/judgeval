# Changelog

## External

*   **Tracing:** Fixed LangGraph tracing issues.
*   **Observability:** Implemented OpenTelemetry support.

## Internal

*   **Caching:** Implemented a caching layer on the application side using Redis.
*   **Asynchronous Migration:** Migrated the codebase from synchronous to asynchronous functions.
*   **Clustering UI:** Improved the UI for clustering to mark successes and failures.
*   **Clustering Performance:** Added caching to improve performance when revisiting clusters.
*   **Self-Hosting:** Improved and updated the self-hosting setup flow.
*   **Infrastructure:** Fully set up staging and production branches/environments for development.
*   **Documentation:** Formalized AWS network diagram for backend infrastructure.
